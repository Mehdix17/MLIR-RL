\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}
Code optimization within modern compiler infrastructures is vital for improving the performance of machine learning models, which continue to grow in size and complexity. Traditional manually crafted heuristics are difficult to scale, rigidly designed, and fall short when adapting to diverse hardware architectures. \gls{drl} provides a compelling alternative by allowing an agent to automatically discover optimal transformation sequences through trial and error.

In this work, we present an advanced \gls{drl}-based framework for automatic code optimization targeting the \gls{mlir} compiler. To overcome the limitations of prior approaches and handle complex, real-world neural network programs, we introduce several novel mechanisms. First, we adopt a Transformer-based architecture to effectively capture the hierarchical structure of deep loop nests. Second, we integrate hardware-aware feature observations, enabling the agent to learn cross-platform generalization strategies. Additionally, we explored a new reward shaping approach, utilizing a diverse dataset of full neural network models.

Experimental results indicate that our proposed agent achieves noticeable performance improvements and speedups over the standard \gls{mlir} heuristics Base across various hardware platforms. By effectively learning adaptive, hardware-aware optimization policies, this work demonstrates the increasing potential of \gls{drl}-guided compilation for modern deep learning workloads.

\vspace{0.5cm}
\noindent \textbf{Keywords:} Deep Reinforcement Learning, Code Optimization, \gls{mlir}, Neural Networks, Compilers.

% ---- Résumé ----
\chapter*{Résumé}
\addcontentsline{toc}{chapter}{Résumé}
L'optimisation de code au sein des infrastructures de compilation modernes est vitale pour améliorer les performances des modèles d'apprentissage automatique, dont la taille et la complexité ne cessent de croître. Les heuristiques traditionnelles conçues manuellement sont difficiles à mettre à l'échelle, de conception rigide et peinent à s'adapter à diverses architectures matérielles. L'apprentissage par renforcement profond (\gls{drl}) offre une alternative convaincante en permettant à un agent de découvrir automatiquement des séquences de transformations optimales par essais et erreurs.

Dans ce travail, nous présentons un système avancé basé sur le \gls{drl} pour l'optimisation automatique du code dans le cadre du compilateur \gls{mlir}. Afin de surmonter les limites des approches précédentes et de traiter des programmes de réseaux de neurones complexes et réels, nous introduisons plusieurs mécanismes novateurs. Premièrement, nous adoptons une architecture basée sur les Transformers pour capturer efficacement la structure hiérarchique des boucles imbriquées profondes. Deuxièmement, nous incluons des observations de caractéristiques tenant compte des caractéristiques matérielles, permettant à l'agent d'apprendre des stratégies de généralisation multiplateformes. De plus, nous explorons une nouvelle approche de récompense personnalisée, tout en utilisant un ensemble de données diversifié de modèles de réseaux de neurones complets.

Les résultats d'expérimentations indiquent que l'agent proposé permet d'obtenir des améliorations d'accélérations "speedups" notables sur plusieurs modèles, opérations et à travers diverses plateformes matérielles. En apprenant efficacement des optimisations adaptatives tenant compte du matériel, ce travail démontre le potentiel croissant de la compilation guidée par le \gls{drl} pour les charges de travail modernes de l'apprentissage profond.

\vspace{0.5cm}
\noindent \textbf{Mots-clés~:} Apprentissage par renforcement profond, Optimisation de code, MLIR, Réseaux de neurones, Compilateurs.
