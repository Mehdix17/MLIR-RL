# MLIR-RL Progress Reporting Runbook (Unified & Accelerated)

Use this skill to fetch, synchronize, and report the current status of all training jobs, evaluation batches, and storage quotas in under 2 seconds.

---

## 🛠️ Step 1. Run Unified Fast Report
Run the fast report script to get training progress, active Slurm jobs, evaluation status, and Lustre quota in a single execution (runs concurrently using background threads):
```bash
python scripts/utils/fast_report.py -d ops_and_blocks
```
*Note: Redirect output to a temporary file and read it to prevent truncation of wide tables.*

---

## 📊 Step 2. Present the Output Report
Directly print the output generated by the script, which is formatted as Markdown tables representing:
1. **Active Slurm Jobs Table**
2. **Training Progress Table**
3. **Evaluation Progress Table**
4. **Lustre Storage Quota Table**
